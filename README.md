# 3. Key Features (주요 기능)

- **퀴즈**:
  - 지도에 표시된 국가의 이름을 맞추는 퀴즈입니다.
  - 정답을 입력하고 제출하면 정답 여부를 확인할 수 있습니다.
- **랭킹**:
  - 퀴즈 점수를 기반으로 랭킹을 확인할 수 있습니다.
- **문화권별 퀴즈**:
  - 전체 국가 외에 특정 문화권에 속한 국가들만 대상으로 퀴즈를 풀 수 있습니다.
